# 📊 DataCo Retail Sales Dashboard

Welcome to my Power BI project! This dashboard provides a comprehensive analysis of retail sales performance, helping businesses make data-driven decisions.

---

## 🚀 Project Overview

This project focuses on analyzing retail sales data using **Power BI**. It highlights key metrics such as:

- Total Sales
- Revenue
- Profit Margin
- Unit Sold
- Employee Performance

The dashboard is designed to give clear insights into sales trends and business performance.
